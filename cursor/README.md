# 燕云心力计算器

这是一个用于计算燕云心力恢复时间的工具，提供命令行版本和Web版本。

## 功能说明

### 参数设置
- **心力上限**：600点
- **恢复速度**：每8分钟恢复1点

## 🌐 Web版本（推荐）

### 安装依赖

```bash
pip install -r requirements.txt
```

### 启动Web服务器

```bash
python app.py
```

### 访问网站

在浏览器中打开：`http://localhost:5000`

### Web版本特点

- ✨ 美观的现代化界面
- ⏰ 自动显示当前系统时间
- 📱 响应式设计，支持手机访问
- 🎯 实时计算结果
- 💡 简单易用的操作界面

## 💻 命令行版本

### 使用方法

1. 运行程序：
   ```bash
   python yan_yun_xinli.py
   ```

2. 按提示输入：
   - 当前心力点数（0-600）

3. 查看结果：
   - 程序会自动获取系统时间
   - 显示需要恢复的点数
   - 恢复所需的时间
   - 下次满心力的具体时间

### 示例

```
当前系统时间：14:30:00
请输入当前心力点数（0-600）：450

计算结果：
当前心力：450点
当前时间：14:30:00
需要恢复：150点
恢复时间：1200分钟 (20小时0分钟)
下次满心力时间：10:30:00
```

## 📐 计算公式

- 需要恢复点数 = 600 - 当前心力
- 恢复时间（分钟）= 需要恢复点数 × 8
- 满心力时间 = 当前时间 + 恢复时间
