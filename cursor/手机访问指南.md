# 手机访问网站指南

## 📱 你的电脑IP地址

**主要IP地址：192.168.1.5**

**手机访问地址：http://192.168.1.5:5000**

## ✅ 访问步骤

### 1. 确保服务器正在运行

在电脑上运行：
```bash
python app.py
```

你应该看到类似这样的输出：
```
 * Running on http://0.0.0.0:5000
```

### 2. 确保手机和电脑在同一WiFi网络

- 手机和电脑必须连接到**同一个WiFi路由器**
- 不能使用手机热点（除非电脑连接手机热点）

### 3. 在手机浏览器中访问

打开手机浏览器（Chrome、Safari等），输入：
```
http://192.168.1.5:5000
```

### 4. 如果无法访问，检查防火墙

#### Windows防火墙设置：

1. 打开"Windows Defender 防火墙"
2. 点击"高级设置"
3. 选择"入站规则" → "新建规则"
4. 选择"端口" → 下一步
5. 选择"TCP"，输入端口号：`5000`
6. 选择"允许连接"
7. 应用到所有配置文件
8. 命名为"Flask Web Server"

或者临时关闭防火墙测试（不推荐，仅用于测试）

## 🔧 常见问题

### 问题1：连接超时
- **解决**：检查服务器是否正在运行
- **解决**：确认IP地址是否正确（运行 `python get_ip.py` 查看）

### 问题2：无法访问此网站
- **解决**：检查手机和电脑是否在同一WiFi
- **解决**：检查防火墙设置

### 问题3：IP地址变化
- **解决**：每次重启路由器后，IP可能变化
- **解决**：重新运行 `python get_ip.py` 获取新IP

## 💡 快速测试

1. 在电脑浏览器访问：`http://localhost:5000` （应该能正常访问）
2. 在电脑浏览器访问：`http://192.168.1.5:5000` （应该也能访问）
3. 如果步骤2失败，说明防火墙阻止了外部访问
4. 如果步骤2成功，手机应该也能访问

## 📝 注意事项

- IP地址可能会变化（如果路由器重启）
- 确保Flask服务器持续运行
- 如果修改了代码，需要重启服务器
